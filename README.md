# Brian — AI Internship Search Assistant

Brian is a personalized AI assistant workspace configured to support a college student's internship search.

## Purpose

This project was created to help manage and improve the internship application process through:

- resume iteration and editing
- internship search support
- application tracking and follow-up
- persistent memory/context across sessions
- structured workspace configuration for long-term assistance

## What I Built

- Defined the assistant's identity, mission, and memory structure
- Configured workspace files for long-term context and continuity
- Designed the assistant around a focused use case: helping Patrick Carroll land a summer internship
- Used prompt design, structured documentation, and tool-oriented workflows to turn a general-purpose AI assistant into a targeted career-support system

## Skills Demonstrated

- prompt engineering
- AI workflow design
- documentation and configuration management
- human-AI interaction design
- iterative product thinking

## Notes

This repository documents the project concept and workspace design used to create Brian as a focused AI assistant for internship support.
